The `<checkbox>` element is a control that allows the user to select a binary state, either checked or unchecked.

### Part Definitions

`::host` — Root element that contains the individual control and label.

`::part-checkbox` — The checkbox control accepting and displaying the state of the checkbox.

`::part-checked-indicator` — The element indicating the checked state of the checkbox.

`::part-indeterminate-indicator` — element indicating the indeterminate state of the checkbox.

`::part-label` — The text-based label of the checkbox.

### Events

- `::host` Dispatches a `change` event when `checked` is changed.
- `::host` Dispatches an `input` event when `checked` is changed.

## Accessibility

### Keyboard

Keyboard users can operate the `<checkbox>` as follows:

- Focus on the `<checkbox>` element using the `Tab` key. The control can be
  focused by default by setting the `autofocus` attribute.
- Having focus, users can:

  - Change the `checked` state by pressing the "Space" key. The new state should be programtically visually reflected by the checkbox

  ### Assistive Technology

Implements the [checkbox role](https://www.w3.org/TR/wai-aria-1.1/#checkbox) per the
[HTML AAM spec](https://www.w3.org/TR/html-aam-1.0/#html-element-role-mappings).

## States / Properties

### &ltcheckbox&gt

- **checked** (_bool_): Controls whether the checkbox is checked or unchecked.

- **defaultChecked** (_bool_): The initial value for `checked`. Defaults to false.

- **value** (_string_): The value of the checkbox.

- **autofocus** (_bool_): Get focus by default.

- **disabled** (_bool_): Prevents the user from interacting with the control. Defaults to false.

- **form** (_string_): Associates the element with a form in the document whose `id` is this value.

- **name** (_string_): Specify the name of the control for purposes of form submission.

- **required** (_bool_): Indicates that the checkbox is invalid unless checked.

- **readonly** (_bool_): Indicates that the checkbox is not interactve but it's value should still be submitted with the form.
